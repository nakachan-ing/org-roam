#+title: 【Index】AWS-Lambda

* AWS Lambdaの概要
AWS Lambdaは、コードをサーバレスで実行するためのサービス。

* ポイント
- サーバの構築や管理をAWSに任せることができる。
- 開発者はアプリケーションの開発に集中できる。
